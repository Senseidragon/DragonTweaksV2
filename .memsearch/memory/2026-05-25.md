## Test Chunk — Keyword Retrieval Verification

**Title:** Event bus routing must not be mixed  
**Type:** Architecture rule  
**Enforcement level:** Hard — mixing silently fails, no runtime error  
**Intent triggers:** event bus, mod bus, game bus, FML setup, NeoForge.EVENT_BUS, ServerStartingEvent, ModConfigEvent  
**Rule or fact:** Mod lifecycle events (FMLCommonSetupEvent, FMLClientSetupEvent, ModConfigEvent) must be registered on the mod bus. Game/world events (ServerStartingEvent, registration events) must be registered on the game/Forge bus (NeoForge.EVENT_BUS). Registering a mod-bus event on the game bus, or vice versa, silently fails — no exception is thrown and the handler is never called.
